# Elite

Casino-style PHP demo with login, wallet progress, favourites, latest bets, and Blackjack.

## Project structure

- `index.php` - home page
- `pages/` - site pages such as Blackjack and favourites
- `api/` - AJAX/PHP endpoints
- `includes/` - shared PHP files
- `assets/css/` - stylesheets
- `assets/js/` - browser JavaScript
- `assets/img/` - images
